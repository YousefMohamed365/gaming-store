# 🎮 Gaming STORE - E-Commerce Platform

A dynamic, fully responsive E-Commerce web application tailored for gamers. Built from scratch using modern front-end technologies to deliver a smooth and immersive user experience with a vibrant dark theme.

---

## 🚀 Features

- **Interactive Authentication:** A clean login gate that captures the user's name and triggers a personalized header greeting upon successful submission.
- **Dynamic Catalog Rendering:** Products and prices are fetched asynchronously from an external data source (`products.json`) and injected seamlessly into the DOM.
- **Detailed Game Modals:** Clicking on "Purchase details" opens an overlay displaying high-quality game art, descriptions, and structural checkout pathways.
- **Live Shopping Cart Tracker:** Implemented a real-time tracking algorithm that monitors selected game state IDs and dynamically increments the global navigation cart counter instance.

---

## 🛠️ Tech Stack

- **HTML5:** Structured semantic markup, form validations, and embedded overlay layouts.
- **CSS3:** Styled with custom responsive grids, flexbox alignments, smooth transitions, and a gaming-focused neon-dark aesthetics.
- **JavaScript (ES6+):** Utilizes DOM/BOM manipulation, state management arrays, asynchronous `fetch` APIs, and modular structural event listeners.

---

├── style.css           # Custom styling guidelines & theme sheets
